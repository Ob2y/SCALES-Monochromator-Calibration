# SCALES-Monochromator-Calibration
Python scripts for laboratory calibration and spectral analysis of the SCALES monochromator for exoplanet spectroscopy
* Objective: Validate wavelength accuracy ($<1.15$ scaling factor correction) and transmission response.
* Instruments: Monochromator calibration for the Keck Observatory.
* Data Range: Analysis of fidelity across the 1.8-2.5 µm range.
Citation: https://www.spiedigitallibrary.org/conference-proceedings-of-spie/13627/3065860/Testing-and-characterization-of-monochromator-for-SCALES-calibration-unit/10.1117/12.3065860.full
